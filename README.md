A minimalist, responsive personal portfolio website designed to showcase my journey, projects, and skills as an aspiring AI Engineer. Built with a high-contrast "tech-enthusiast" dark theme featuring charcoal backgrounds and electric blue accents.

🌐 **Live Website:** [harisankar.com](https://harisankar.com) *(Update this link if using a custom domain, or use https://harisankar005.github.io)*

---

## ⚡ Features

* **Tech-Enthusiast Aesthetic:** Custom UI featuring a charcoal black (`#121212`) background, crisp off-white text, and electric blue (`#00e5ff`) interactive elements.
* **Smooth UX:** Implemented native smooth scrolling and subtle `fadeUp` entrance animations for all sections.
* **Responsive Design:** Fully fluid CSS Grid and Flexbox layouts ensure the site looks flawless on desktop, tablet, and mobile devices.
* **Zero Dependencies:** Built entirely with vanilla HTML5 and CSS3. No heavy frameworks or JavaScript libraries required.

---

## 🛠️ Sections Overview

1. **Hero:** High-impact introduction with quick links to GitHub and a direct contact form.
2. **About Me:** Career objectives and current focus areas in LLM-driven systems and backend development.
3. **Core Competencies & Skills:** Categorized breakdown of my technical stack (Python, C++, RAG Pipelines, FAISS, APIs).
4. **Projects & Achievements:** Highlights of my core work, including:
   * **Aegis:** A multi-agent workflow system.
   * **Semantic Retrieval & QA Systems:** High-dimensional embedding similarity engines.
   * **IBM QISKIT Hackathon:** First Prize (First-Year) for a quantum-classical hybrid solution.
5. **Education:** Academic background and performance metrics.
6. **Interests & Travel:** A personal touch showcasing favorite media and recent travels.
